Proyecto Destacado

Cabeza robótica con visión artificial

Descripción

Sistema robótico capaz de detectar rostros mediante una cámara y un modelo de visión artificial basado en MediaPipe, controlando sus motores para seguirlos en tiempo real.

Tecnologías

  • Python (MediaPipe, visión artificial)
  • Arduino (C/C++)
  • Electrónica
  • Diseño y impresión 3D
  • Servomotores y motores paso a paso
  • Webcam

Funcionamiento:

  • Detección de rostro mediante visión artificial (Ordenador externo con Python)
  • Cálculo de coordenadas del rostro
  • Comunicación en serie Python - Arduino
  • Control de motores en tiempo real basado en el resultado de los cálculos realizados

Posibles mejoras futuras

  • Integración de una Raspberry Pi para poder tener todo el sistema centralizado en el robot
  • Añadir un control PID para mayor suavidad y precisión en el movimiento del robot
  • Crear una interfaz de control para que el usuario pueda utilizar fácilmente el robot
